# 🏛️ The Agentic Trivium: A Framework for Living Governance

![Header](https://raw.githubusercontent.com)

> **"A stale contract is worse than no contract."**

This is a **platform-agnostic boilerplate** for implementing **Contract-Style Comments (CSC)** and the **Missing Axiom** theory. It is designed to ground stateless AI agents in present-tense law, architectural intent, and operational truth.

## 🛠️ The Triumvirate (Core Artifacts)

This repository enforces a "Required Reading Order" to prevent contextual drift and "confident guessing" by AI agents.

1.  **CONTRACT.md (The Law)**: Defines present-tense invariants and logical boundaries. If it must not break, it lives here.
2.  **WHY.md (The Reasoning)**: Ties the artifacts together. It explains the relationship between files and the "Narrowest-Scope Update Rule."
3.  **QUICKSTART.md (The Map)**: Provides a stateless-friendly map of key files and "Proven Checks" to verify behavior.

*Optional: **FUTURE.md** for planned intent, keeping the Contract lean.*

## 🤝 The Agentic Handshake

Unlike static documentation, these are **Living Documents**.

-   **Authorization**: Within a session, the AI agent is authorized and **expected** to act as a Governance Steward.
-   **Responsibility**: If a "Missing Axiom" is discovered or a scope-affecting change occurs, the agent must update the narrowest owning artifact before the session ends.
-   **Verification**: All work must be proven against the checks in `QUICKSTART.md` to ensure the `CONTRACT.md` remains intact.

## 🚀 Getting Started

1.  **Click "Use this template"** to initialize your project.
2.  **Define your Invariants** in `CONTRACT.md`.
3.  **Start your Session** by instructing your agent: *"Read the Trivium in order and adhere to the Governance Ownership policy."*
